Folder Guide - core/pipeline

Purpose
- This folder coordinates reading an image, detecting faces, and applying the chosen anonymization effect.

Files
- processor.py
  Defines the Processor class.

What Processor does
- Loads images from disk with OpenCV
- Uses FaceDetector to detect faces
- Chooses which anonymization function to call based on the requested mode
- Returns:
  processed image array
  detected face list

Main methods
- detect_faces(image_path)
  Reads an image and returns detected Face objects.

- process_image(image_path, selected_ids, mode, options)
  Detects faces and applies blur, pixelate, or mask to the chosen face IDs.

Why this layer exists
- It keeps the route/service layer from needing to know effect-level details.
- It centralizes mode selection in one place.
